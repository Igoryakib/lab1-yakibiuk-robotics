# Lab 1: Building Robot in Gazebo


### 1. Clone Repository

```bash
cd ~
git clone https://github.com/RybOlya/robotics_lpnu.git
cd robotics_lpnu
```

### 2. Build Docker Image

```bash
./scripts/cmd build-docker
```

This takes 10-15 minutes on first run.

### 3. Run Container

```bash
./scripts/cmd run
```
### 4. Install and open VScode

```bash
code .
```
